
This Folder contains the following sub folders

1) Docs : Contains the documentation about installing the MCS7717Ce driver

2) MCS7717CE_Cab : Contains the .Cab file for installing the MCS7717 device
   driver on Windows CE .NET machine

3) drvdsk: Contains the Inf file and MCS7717Ce.dll.

4) Hub Modifications : Contains the documentation about required modifications 
to make MCS7717 device to work under WinCE5.0

5) PrinterTest : Contains the Test_LPT source ( Test application for Parallel Port )
and documentation about how to add this applcation to the OS and test the MCS7717 
Paraellel port
